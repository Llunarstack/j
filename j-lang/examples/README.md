# J Language Examples

Code examples demonstrating J language features.

## Directory Structure

- **basic/** - Basic language features (variables, functions, control flow)
- **advanced/** - Advanced features (classes, traits, async)
- **modules/** - Module system examples
- **tests/** - Test files for development

## Running Examples

```bash
# From j-lang directory
cargo run -- run examples/basic/hello.j

# Or with installed binary
j run examples/basic/hello.j
```

## Example Categories

### Basic
- Variables and types
- Functions
- Control flow
- Collections

### Advanced
- Classes and OOP
- Pattern matching
- Generators
- Error handling

### Modules
- Importing modules
- Creating modules
- Module organization

## Contributing Examples

When adding examples:
1. Place in appropriate category directory
2. Add clear comments
3. Keep examples focused and simple
4. Test before committing
