Control flow

Conditionals, loops, and error handling in Jade.

if / else

if condition {
  // then
} else if other {
  // else if
} else {
  // else
}

One-liner: if cond : expr.

match

match expr {
  case pattern1 > body1
  case pattern2 > body2
  else > default_body
}

cond / when

Multi-branch on a value. cond value { case v > body ... }, when value { case v > body ... }.

unless

unless condition { body } — run body when condition is false. Optional else.

either

either expr { |> true : body1 |> false : body2 } — branch on boolean.

switch

switch expr { case a > body1 case b > body2 }.

while

while condition {
  body
}

for

for x in iterable { body }
for index, x in iterable { body }   # indexed

Iterables: lists, ranges, strings (chars), dict keys/values. Optional modifiers: for x in list rev (reverse), step n, until cond, where cond, zip, parallel, chunked n, windowed n.

return / break / continue

return or return value. break and continue in loops.

defer

defer expr — run expr when leaving the current scope.

try / catch / finally

try {
  risky()
} catch err {
  handle(err)
} finally {
  cleanup()
}

Optional catch var to bind the error. expr? for try-expression.

panic

panic(message) — abort with error message.

guard return

guard condition > return value — early return when condition is true.
