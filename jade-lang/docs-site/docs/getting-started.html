Getting started

Run Jade from the command line: execute a file, start the REPL, or build a native binary.

Installation

  • Windows: Run jade-1.0.0-windows-x86_64-setup.exe from releases, or cargo build --release in jade-lang. Binary: target\release\jade.exe.
  • macOS / Linux: From repo root, cd jade-lang && cargo build --release. Binary: target/release/jade. Optional: installers/linux/install.sh or installers/macos/install.sh.

Run a file

jade file.jdl
jade run file.jdl

Use jade - to read source from stdin (e.g. pipe from editor).

REPL

jade repl
jade idle   # IDLE-style interactive shell

Check syntax

jade check file.jdl

Build native binary (AOT)

jade build file.jdl -o myapp
jade build file.jdl --release -o myapp

Requires LLVM for full AOT support on your platform.

# Built-in algo + typed fn · string interpolation
list: a = [-2, 1, -3, 4, -1, 2, 1]
out("Max subarray: {kadane(a)}")

fn | fib ( int | n ) > {
  if n <= 1 { return n }
  return fib(n - 1) + fib(n - 2)
}
out("fib(10) = {fib(10)}")
Output
Max subarray: 6
fib(10) = 55
