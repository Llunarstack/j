Methods on values

Lists: .len, .push, .pop, .append, .insert, .remove, index/slice. Dicts: .keys, .values, .get, .set. Strings: indexing, slice, .contains (with regex when available). Ranges: .len. Built-in map, filter, reduce work on iterables.
