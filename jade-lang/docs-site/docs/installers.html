Installers

Download Jade for Windows, macOS, and Linux. Each release version has its own section so you can pick a specific version or the latest.

Version 1.0.0

Current stable release. Choose your platform below.

Windows

64-bit (x86_64). Run the setup for a guided install, or use the portable zip to run without installing.

Setup (EXE) Portable (ZIP)

After install, run jade file.jdl or jade repl from the command line. See Getting started for usage.

macOS

Universal (x86_64 and Apple Silicon). Use the install script or the disk image.

macOS (tar.gz) macOS x86_64 (PKG) macOS Apple Silicon (PKG)

Or build from source: cd jade-lang && cargo build --release, then run ./installers/macos/install.sh.

Linux

64-bit (x86_64). Choose a package format for your distribution.

Debian / Ubuntu (.deb) Fedora / RHEL (.rpm) Portable (tar.gz)

Or use the install script from the repo: ./installers/linux/install.sh after building.

From source

Build the latest from the repository. Requires Rust.

git clone https://github.com/Llunarstack/j.git
cd j/jade-lang
cargo build --release

Binary: target/release/jade (or jade.exe on Windows). For install scripts and PATH setup, see Getting started → Installation.

Other versions

Older or pre-release versions are listed on GitHub Releases. This page will add dedicated sections for future stable versions (e.g. 1.1.0, 2.0.0) as they are released.
