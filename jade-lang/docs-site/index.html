  if n <= 1 { return n }
  return fib(n - 1) + fib(n - 2)
}

Output

Why Jade?

One language for scripts, tools, and small binaries. Built-in algorithms and data structures. No runtime to install—just one binary. Pipe-first syntax and optional types keep code short and clear.

Syntax that scales

Pipe-style fn | name (params) > body and optional types: (x, y) or (int | a, str | b). Less boilerplate, same clarity when you need it.

Batteries included

DSA and algorithms built in: kadane, fibonacci, graphs, Union-Find, trie, crypto, regex. Plus out, range, map, filter, and 100+ more. No hunting for libs.

One binary, every platform

Run files, REPL, or build a native executable. jade file.jdl, jade repl, jade build file.jdl -o app. No separate runtime or interpreter install.

Cool features in one place

Match expressions, comprehensions, pipelines, graphs, crypto, and 100+ built-ins. One reference page.

# Optional types · interpolation · pipeline
fn | double (x) > { return x * 2 }
int: n = 21
out(n |> double)
out("Sum: {n}")

# List comprehensions · built-in algo
list: sq = [x * x for x in [1, 2, 3, 4]]
out(kadane([-2, 1, -3, 4, -1, 2, 1]))

42

Install

Windows, macOS, or from source.

# Windows — run installer or portable zip
jade-1.0.0-windows-x86_64-setup.exe

# From source (all platforms)
git clone https://github.com/Llunarstack/j
cd j/jade-lang
cargo build --release
# Binary: target/release/jade  (or jade.exe on Windows)
