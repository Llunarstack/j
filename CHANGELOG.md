# Changelog

All notable changes to the J programming language will be documented in this file.

## [Unreleased]

### Added
- Full module system with file loading and caching
- Trait system with method signatures and default implementations
- Async/await support with Future values
- Task and channel concurrency primitives
- Generator functions
- Counter arithmetic operations
- Grid enhancements (neighbors8, find_all, row, col)
- VS Code extension with syntax highlighting
- Multi-platform installers (Windows, macOS, Linux)

### Changed
- Improved parser error messages
- Enhanced type system
- Optimized interpreter performance

### Fixed
- Module import resolution
- Class instantiation bugs
- Pattern matching edge cases
